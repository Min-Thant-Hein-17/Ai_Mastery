# Prompt Engineering

Four frameworks for the communication with Ai: 

1.TAG: Define your Task, Action, and Goal

2.CARE: Add Context, Action, Result, and Example

3.RACE: Specify Role, Action, Context, and Expectation

4.RISE: Combine Role, Input, Steps, and Expectation
